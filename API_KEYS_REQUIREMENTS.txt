⚙️ ENVIRONMENT SETUP INSTRUCTIONS
================================================================================

1. CREATE ENVIRONMENT FILE:
   touch .env.local

2. ADD VARIABLES (example format):
   VITE_GOOGLE_MAPS_API_KEY=AIzaSyC-your-actual-key-here
   VITE_YELP_API_KEY=Bearer your-yelp-key-here
   VITE_FIREBASE_API_KEY=AIzaSyD-your-firebase-key-here

3. ENSURE .GITIGNORE INCLUDES:
   .env.local
   .env
   .env.production

4. RESTART DEVELOPMENT SERVER:
   npm run dev

================================================================================
🛡️ SECURITY BEST PRACTICES
================================================================================

✅ DO:
- Use VITE_ prefix for client-side environment variables
- Keep sensitive keys on server-side only
- Rotate API keys regularly
- Set up API key restrictions (HTTP referrers, IP addresses)
- Monitor API usage and set up billing alerts
- Use different keys for development/staging/production

❌ DON'T:
- Commit API keys to version control
- Share API keys in chat/email
- Use production keys in development
- Expose server-side keys to client
- Use keys without restrictions
